# Code of Conduct

Open Social Network is intended to support an open, interoperable social web.

Participation in this project requires respectful, constructive behavior. Harassment, threats, personal attacks, or abuse are not acceptable in issues, pull requests, discussions, or project spaces.

Maintainers may remove disruptive content and restrict participation when necessary to protect the project and its contributors.

## Community Context

Open Social Network explores sovereign profiles, replaceable aggregators, portable audiences, and user-facing simplicity. That direction raises real tradeoffs around moderation, spam, safety, discovery, scaling, onboarding, and creator incentives.

Discuss those tradeoffs directly and respectfully. Treat the project as public infrastructure: be specific, evidence-oriented, and open to different assumptions about how social systems should work.
